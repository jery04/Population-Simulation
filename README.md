# 🧬 Human Population Simulation

A discrete-event population simulator that models demographic dynamics with life events and age-based probabilities.

## Overview 📘

This project simulates population evolution over time by modeling individuals with realistic life cycles. Each person has attributes such as age, sex, partnership status, and fertility preferences. The simulator processes stochastic events (deaths, partnerships, breakups, pregnancies) on a continuous timeline and records population size over time.

Key ideas ✨:

- Individuals are modeled as agents with ages in years and evolving social states .
- Events happen on a stochastic schedule (exponential time steps) rather than fixed monthly ticks.
- Probabilities are driven by age-range tables for mortality, pregnancy, relationship desire, and partner matching.

## Repository structure 🧭

Top level 📂:

- `main.py`: Command-line entry point that runs multiple simulations and saves a chart.
- `requirements.txt`: Python dependencies.
- `README.md`: Project documentation.
- `results/`: Output artifacts (charts saved by default).
- `scripts/`: Core simulation modules.

Scripts 🛠️:

- `scripts/person.py`: `Person` data model (age, sex, partner, fertility state).
- `scripts/sampler.py`: Random sampling utilities and exponential time-step helpers.
- `scripts/simulation.py`: `Simulador` engine, event scheduling, and event handlers.
- `scripts/statistics_plot.py`: Multi-run statistics, plotting, and output path handling.
- `scripts/table.py`: Probability tables and age-range lookup function.

Results ✅:

- `results/population_growth_standard_deviation.png`: Default output chart generated by `main.py`.
- `results/population_growth_confidence_interval.png`: Auxiliary confidence interval chart generated for multi-run output.
- `results/population_growth_births_confidence_interval.png`: 95% confidence interval for cumulative births.
- `results/population_growth_deaths_confidence_interval.png`: 95% confidence interval for cumulative deaths.
- `results/population_growth_pairs_confidence_interval.png`: 95% confidence interval for cumulative pairs formed.
- `results/population_growth_breaks_confidence_interval.png`: 95% confidence interval for cumulative breakups.

## Dependencies 📦

- Python 3.8 or newer
- `matplotlib` (see `requirements.txt`)

Install dependencies:

```bash
pip install -r requirements.txt
```

## How to run (step by step) 🚀

1. Open a terminal at the project root.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the default simulation + chart:

```powershell
py main.py
```

Or on macOS/Linux:

```bash
python main.py
```

4. Check the output file:

```
results/population_growth_standard_deviation.png
results/population_growth_confidence_interval.png
```

The console will also print the mean final population across all runs 📌.

## Configuration and customization 🧪

Edit `main.py` to change the default run parameters:

- `male_count` and `female_count`: initial population.
- `years`: simulated years.
- `runs`: number of independent runs for statistics.
- `sigma`: width of the standard deviation band.
- `plot_all`: plot every run in gray instead of a sigma band.
- `show`: display the chart interactively.
- `output`: output filename (saved under `results/` when relative).

You can also call the plotting function programmatically 🧑‍💻:

```python
from pathlib import Path
from scripts.statistics_plot import build_chart, resolve_output_path

output = resolve_output_path(Path("custom_chart.png"))
mean_final_pop = build_chart(
    male_count=120,
    female_count=130,
    years=50,
    output=output,
    runs=30,
    sigma=1,
    plot_all=False,
    show=False,
)
print(mean_final_pop)
```

## Notes 🧾

- The simulation is stochastic; results will vary run to run 🎲.
- Probability tables live in `scripts/table.py` if you want to explore different demographic scenarios 🔬.
